Projects

Word Unscrambler

A web app that takes a word jumble and attempts to figure out what word the letters form. It categorizes the results based on likelihood of a successful word, with confirmed results being matched against a dictionary of words. The backend uses Python and mySQL (database) and pages are served using Flask templates. Additional input validation is performed with Javascript and it is styled using Bootstrap.

Live Site Repository (Github)

Book Review Site

A mock book review site that allows users to search for books by title/author/ISBN. It allows authenticated users to rate and review books, with ratings fetched using Goodread's api. The site itself also provides api access via json. The backend uses Flask (Python) and the book information uses PostgreSQL, with the entire site being hosted on Heroku. Project was created for Harvard's CS50W course.
